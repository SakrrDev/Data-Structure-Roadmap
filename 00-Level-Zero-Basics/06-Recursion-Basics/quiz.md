# Data Structures Quiz (MCQs)

## Instructions
- Choose the correct answer for each question.
- Questions are ordered from easy to hard.
- Do NOT check answers until you finish the quiz.

---

### Questions

1. What is recursion?

   * a) Looping using for/while
   * b) A function calling itself
   * c) A conditional statement
   * d) Sorting algorithm

2. What is the purpose of a base case?

   * a) To repeat the function
   * b) To stop recursion
   * c) To increase recursion depth
   * d) To initialize variables

3. Recursive case is:

   * a) The stopping condition
   * b) The function calling itself with modified input
   * c) A variable declaration
   * d) Ignored during recursion

4. Which is an example of direct recursion?

   * a) Function A calls Function B, which calls Function A
   * b) Function calls itself directly
   * c) Loop inside a function
   * d) Sorting function

5. Which is an example of indirect recursion?

   * a) Function calls itself
   * b) Function A calls B, B calls A
   * c) Loop inside function
   * d) Conditional statement

6. Tail recursion means:

   * a) Recursive call is first operation
   * b) Recursive call is last operation
   * c) No recursive call
   * d) Recursion with multiple calls

7. Non-tail recursion means:

   * a) Recursive call is last operation
   * b) Recursive call is not last operation
   * c) Function calls no one
   * d) Multiple functions call each other

8. Factorial of n is usually implemented using:

   * a) Tail recursion
   * b) Non-tail recursion
   * c) Loop only
   * d) Iteration only

9. Fibonacci sequence can be implemented using:

   * a) Recursion
   * b) Iteration
   * c) Both a and b
   * d) None

10. Stack overflow can occur if:

    * a) Base case missing or too deep recursion
    * b) Base case exists
    * c) Function is iterative
    * d) Only loops used

11. Recursive function must have:

    * a) Loop
    * b) Base case
    * c) Print statement
    * d) Multiple returns

12. Advantages of recursion include:

    * a) Simpler code
    * b) Easier to read
    * c) Divide-and-conquer problems
    * d) All of the above

13. Disadvantages of recursion include:

    * a) Function call overhead
    * b) Stack overflow
    * c) Less efficient than iteration sometimes
    * d) All of the above

14. Example of tail recursion is:

    * a) factorial(n)
    * b) sum(n, acc)
    * c) fibonacci(n)
    * d) Towers of Hanoi

15. Example of non-tail recursion is:

    * a) sum(n, acc)
    * b) factorial(n)
    * c) Increment counter
    * d) DFS traversal

16. Recursive solution is most suitable for:

    * a) Simple linear problems
    * b) Problems with repetitive or hierarchical structure
    * c) Only arrays
    * d) Only loops

17. In recursion, parameters usually:

    * a) Stay constant
    * b) Change towards base case
    * c) Are ignored
    * d) Must be global variables

18. Recursive tree traversal example includes:

    * a) Preorder
    * b) Inorder
    * c) Postorder
    * d) All of the above

19. Towers of Hanoi problem is solved using:

    * a) Iteration only
    * b) Recursion
    * c) Sorting
    * d) Searching

20. To prevent infinite recursion:

    * a) Add loops
    * b) Define base case
    * c) Call other functions
    * d) Ignore recursion depth
