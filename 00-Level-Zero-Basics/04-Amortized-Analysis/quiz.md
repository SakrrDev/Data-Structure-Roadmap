# Data Structures Quiz (MCQs)

- Questions are ordered from easy to hard.

### Questions

1. What is amortized analysis used for?

   * a) Worst-case of single operation
   * b) Average running time over sequence of operations
   * c) Best-case time
   * d) Memory usage

2. Amortized analysis is different from average-case analysis because:

   * a) It depends on input distribution
   * b) It gives guarantees regardless of input
   * c) It only works for arrays
   * d) It ignores expensive operations

3. Which of the following is a technique of amortized analysis?

   * a) Aggregate Method
   * b) Binary Search
   * c) Recursion
   * d) Sorting

4. In the aggregate method, how is the amortized cost calculated?

   * a) By maximum cost of any operation
   * b) Total cost divided by number of operations
   * c) Random guess
   * d) By input size only

5. In the accounting method, what is done?

   * a) Charge actual cost only
   * b) Overcharge cheap operations to pay for expensive ones
   * c) Ignore expensive operations
   * d) Calculate best-case

6. The potential method uses:

   * a) A counter
   * b) A potential function representing the state
   * c) Sorting algorithm
   * d) Random number generator

7. Which data structure benefits from amortized analysis?

   * a) Static array
   * b) Dynamic array
   * c) Binary search tree
   * d) Linked list

8. The amortized cost ensures:

   * a) Each operation is cheap
   * b) Average cost per operation is small over sequence
   * c) Worst-case cost is minimized
   * d) Input size is reduced

9. Dynamic array resizing is analyzed using:

   * a) Accounting method
   * b) Aggregate method
   * c) Potential method
   * d) Both b and c

10. In stack multipop, amortized analysis helps because:

    * a) Pops are always cheap
    * b) Pops may be costly but average is small
    * c) Pushes are costly
    * d) It reduces memory

11. Potential function must be:

    * a) Negative
    * b) Zero
    * c) Non-negative
    * d) Random

12. Amortized cost formula in potential method:

    * a) actual cost - ΔΦ
    * b) actual cost + ΔΦ
    * c) ΔΦ only
    * d) actual cost / ΔΦ

13. Which method assigns extra credit to operations?

    * a) Aggregate
    * b) Accounting
    * c) Potential
    * d) Recursive

14. Amortized analysis is most useful when operations are:

    * a) Always cheap
    * b) Occasionally expensive
    * c) Never expensive
    * d) Always worst-case

15. Binary counter increment analysis usually uses:

    * a) Aggregate method
    * b) Accounting method
    * c) Potential method
    * d) Sorting method

16. Which method is easiest to understand conceptually?

    * a) Aggregate method
    * b) Accounting method
    * c) Potential method
    * d) All are equally hard

17. Amortized analysis guarantees:

    * a) Worst-case cost per operation
    * b) Average cost per operation over sequence
    * c) Best-case cost per operation
    * d) Memory usage reduction

18. Stack multipop example demonstrates:

    * a) Aggregate method
    * b) Accounting method
    * c) Potential method
    * d) None

19. Which is NOT a valid amortized analysis technique?

    * a) Aggregate method
    * b) Accounting method
    * c) Potential method
    * d) Quick sort

20. The main purpose of amortized analysis:

    * a) Predict input
    * b) Analyze memory
    * c) Analyze average operation cost over time
    * d) Improve recursion
