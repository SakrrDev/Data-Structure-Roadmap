# Data Structures Quiz (MCQs)

## Instructions
- Choose the correct answer for each question.
- Questions are ordered from easy to hard.
- Do NOT check answers until you finish the quiz.

---

### Questions

1. What is iteration?

   * a) Function calling itself
   * b) Repeated execution using loops
   * c) Conditional statement
   * d) Memory allocation

2. What is recursion?

   * a) Looping
   * b) Function calling itself
   * c) Sorting
   * d) Conditional jump

3. Which is generally faster?

   * a) Iteration
   * b) Recursion
   * c) Both equal
   * d) Depends on variable names

4. Which uses more memory?

   * a) Iteration
   * b) Recursion
   * c) Same
   * d) None

5. Iteration uses:

   * a) Stack per call
   * b) Same stack frame
   * c) Heap memory
   * d) Registers only

6. Recursion requires:

   * a) Loop condition
   * b) Base case
   * c) Iterative statement
   * d) Array

7. Example of iteration:

   * a) sum(n) using function calling itself
   * b) sum using for loop
   * c) factorial recursive
   * d) Fibonacci recursive

8. Example of recursion:

   * a) For loop sum
   * b) factorial(n)
   * c) while loop sum
   * d) simple assignment

9. Iteration is more suitable for:

   * a) Hierarchical problems
   * b) Simple repetitive tasks
   * c) Trees
   * d) Graph traversal

10. Recursion is more suitable for:

    * a) Simple loops
    * b) Problems with natural recursive structure
    * c) Incrementing counters
    * d) Simple addition

11. Base case is required in:

    * a) Iteration
    * b) Recursion
    * c) Both
    * d) Neither

12. Termination of loops is defined by:

    * a) Base case
    * b) Loop condition
    * c) Function call
    * d) Stack frame

13. Tail recursion is:

    * a) Recursive call first operation
    * b) Recursive call last operation
    * c) Loop inside recursion
    * d) Infinite recursion

14. Recursion can cause:

    * a) Stack overflow
    * b) Infinite loops
    * c) Memory leak
    * d) Compile error

15. Iterative code is generally:

    * a) More readable for complex trees
    * b) More efficient for simple repetitive tasks
    * c) Slower than recursion
    * d) Requires stack per call

16. Recursive code is generally:

    * a) Efficient for simple addition
    * b) Elegant for divide-and-conquer problems
    * c) Always faster
    * d) Uses less memory

17. Which can be optimized with tail recursion?

    * a) Iteration
    * b) Recursion
    * c) Both
    * d) Neither

18. Memory usage of iteration is:

    * a) Low
    * b) High
    * c) Variable
    * d) Unlimited

19. Memory usage of recursion is:

    * a) Low
    * b) High due to function calls
    * c) Same as iteration
    * d) Zero

20. Choosing between iteration and recursion depends on:

    * a) Problem nature
    * b) Readability
    * c) Efficiency requirements
    * d) All of the above
