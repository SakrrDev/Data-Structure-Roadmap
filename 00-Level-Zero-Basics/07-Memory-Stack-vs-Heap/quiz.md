### Questions

1. What is the stack memory used for?

   * a) Dynamic memory allocation
   * b) Static memory allocation
   * c) Storing files
   * d) Networking

2. Heap memory is used for:

   * a) Local variables only
   * b) Dynamic memory allocation
   * c) Function calls
   * d) CPU registers

3. Which memory is managed automatically?

   * a) Heap
   * b) Stack
   * c) Disk
   * d) Cache

4. Which memory requires manual deallocation?

   * a) Stack
   * b) Heap
   * c) Registers
   * d) ROM

5. Stack memory follows which order?

   * a) FIFO
   * b) LIFO
   * c) Random
   * d) Circular

6. Heap memory is generally:

   * a) Small
   * b) Large
   * c) Fixed at compile time
   * d) Very fast

7. Which is faster to access?

   * a) Heap
   * b) Stack
   * c) Disk
   * d) Cache

8. Where are function parameters stored?

   * a) Heap
   * b) Stack
   * c) Global memory
   * d) ROM

9. What happens to stack memory when function returns?

   * a) It persists
   * b) It is automatically freed
   * c) It moves to heap
   * d) Undefined

10. Which is at risk of memory leaks if not managed?

    * a) Stack
    * b) Heap
    * c) CPU cache
    * d) Registers

11. Which memory is limited in size?

    * a) Heap
    * b) Stack
    * c) Both
    * d) Disk

12. Stack memory allocation is:

    * a) Dynamic
    * b) Static
    * c) Random
    * d) Networked

13. Heap memory allocation is:

    * a) Static
    * b) Dynamic
    * c) Fixed
    * d) CPU-only

14. Local variables in a function are stored in:

    * a) Heap
    * b) Stack
    * c) Disk
    * d) Global memory

15. Which memory can grow or shrink during runtime?

    * a) Stack
    * b) Heap
    * c) Registers
    * d) ROM

16. Risk of stack overflow occurs when:

    * a) Too many heap allocations
    * b) Too deep recursion
    * c) Global variables used
    * d) Disk is full

17. Heap memory deallocation uses which keyword in C++?

    * a) free
    * b) delete
    * c) pop
    * d) push

18. Stack memory allocation happens when:

    * a) Program starts
    * b) Function is called
    * c) Object is created with new
    * d) Variable is static

19. Heap memory lifetime is:

    * a) Until function returns
    * b) Until manually freed
    * c) Always temporary
    * d) Always fixed

20. Correct usage of stack and heap helps to:

    * a) Reduce compilation time
    * b) Improve performance and prevent errors
    * c) Avoid syntax errors
    * d) Increase file size
