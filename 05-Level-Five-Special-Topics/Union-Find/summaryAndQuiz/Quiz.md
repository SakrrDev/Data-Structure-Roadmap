


# Union-Find (Disjoint Set Union) – MCQs

## 1. What is the primary purpose of the **Union-Find** data structure?
A. To sort elements efficiently  
B. To manage dynamic arrays  
C. To keep track of disjoint sets  
D. To store hierarchical data  

---

## 2. Which two main operations are supported by Union-Find?
A. Push and Pop  
B. Insert and Delete  
C. Find and Union  
D. Search and Sort  

---

## 3. What does the **Find** operation return?
A. The size of the set  
B. The parent of a node  
C. The representative (root) of a set  
D. All elements in the set  

---

## 4. Which optimization technique helps reduce the time complexity by flattening the structure?
A. Union by Rank  
B. Path Compression  
C. Breadth-First Search  
D. Depth-First Search  

---

## 5. Which technique attaches the smaller tree under the larger one during union?
A. Path Compression  
B. Backtracking  
C. Union by Rank / Size  
D. Recursion  

---

## 6. What is the **amortized time complexity** of Union-Find operations with both optimizations?
A. O(1)  
B. O(log n)  
C. O(n)  
D. O(α(n))  

---

## 7. Union-Find is commonly used in which of the following problems?
A. Binary Search  
B. Shortest Path Algorithms  
C. Cycle Detection in Graphs  
D. Tree Traversal  

---

## 8. In **Kruskal’s Algorithm**, Union-Find is used to:
A. Sort edges  
B. Detect cycles while building the MST  
C. Store adjacency lists  
D. Perform BFS  

---

## 9. Initially, how many sets are there in a Union-Find structure with `n` elements?
A. 0  
B. 1  
C. n  
D. n²  

---

## 10. Which of the following is NOT a typical application of Union-Find?
A. Dynamic connectivity  
B. Image segmentation  
C. Cycle detection  
D. Array sorting  

---

## Answers

1. **C. To keep track of disjoint sets**  
2. **C. Find and Union**  
3. **C. The representative (root) of a set**  
4. **B. Path Compression**  
5. **C. Union by Rank / Size**  
6. **D. O(α(n))**  
7. **C. Cycle Detection in Graphs**  
8. **B. Detect cycles while building the MST**  
9. **C. n**  
10. **D. Array sorting**
