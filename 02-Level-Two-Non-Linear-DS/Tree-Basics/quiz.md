# Tree Basics Quiz

## Multiple Choice Questions

1. What is the root of a tree?
   - A) Leaf node
   - B) Topmost node
   - C) Node with children
   - D) Edge

2. What is a leaf node?
   - A) Node with no children
   - B) Root node
   - C) Node with children
   - D) Edge

3. In a tree, what is a node called that has children?
   - A) Leaf
   - B) Root
   - C) Parent
   - D) Sibling

4. What is an edge in a tree?
   - A) A child node
   - B) Connection between nodes
   - C) Root node
   - D) Leaf node

5. How is the level of a node defined?
   - A) Number of edges from the root to the node
   - B) Number of children
   - C) Height of the tree
   - D) Leaf nodes count

6. What is the height of a tree?
   - A) Number of nodes
   - B) Number of edges on the longest path from root to a leaf
   - C) Number of children
   - D) Depth of root

7. What is a sibling node?
   - A) Node with no children
   - B) Node with a parent in common
   - C) Root node
   - D) Edge

8. What is a subtree?
   - A) Entire tree
   - B) Tree formed from a node and all its descendants
   - C) Leaf node
   - D) Node with no children

9. Name one example of a hierarchical data structure in real life.
   - A) Linked list
   - B) Array
   - C) File system
   - D) Queue

10. How many children can a binary tree node have at most?
    - A) 1
    - B) 2
    - C) 3
    - D) Unlimited

11. What is a full binary tree?
    - A) Every node has 0 or 2 children
    - B) Every node has 1 child
    - C) Every node has 2 or 3 children
    - D) All nodes are leaves

12. What is a complete binary tree?
    - A) All levels filled except possibly the last, filled from left to right
    - B) Every node has 0 or 2 children
    - C) Only root exists
    - D) Skewed tree

13. What is a skewed tree?
    - A) Every node has 2 children
    - B) Tree with only one child per node
    - C) Full binary tree
    - D) Leaf-only tree

14. What traversal order is Root → Left → Right?
    - A) Inorder
    - B) Preorder
    - C) Postorder
    - D) Level Order

15. What traversal order is Left → Root → Right?
    - A) Inorder
    - B) Preorder
    - C) Postorder
    - D) Level Order

16. What traversal order is Left → Right → Root?
    - A) Inorder
    - B) Preorder
    - C) Postorder
    - D) Level Order

17. What traversal uses a queue to visit nodes level by level?
    - A) Preorder
    - B) Inorder
    - C) Postorder
    - D) Level Order

18. Which traversal of a binary search tree gives sorted output?
    - A) Preorder
    - B) Inorder
    - C) Postorder
    - D) Level Order

19. Time complexity of visiting all nodes in a tree is?
    - A) O(1)
    - B) O(log n)
    - C) O(n)
    - D) O(n^2)

20. If a tree has 7 nodes, and the root is at level 0, what is the maximum possible height?
    - A) 3
    - B) 5
    - C) 6
    - D) 7
