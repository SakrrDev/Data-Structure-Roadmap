# Binary Search Tree (BST) Quiz

## Multiple Choice Questions

1. What is a Binary Search Tree (BST)?
   - A) Any binary tree
   - B) Left child < Node < Right child
   - C) Tree with only left children
   - D) Tree with only right children

2. What property must a BST satisfy?
   - A) Left child > Node < Right child
   - B) Left child < Node < Right child
   - C) Root only
   - D) No property

3. Can a BST have duplicate values?
   - A) Yes
   - B) No
   - C) Only in leaves
   - D) Only in root

4. Where is a smaller value inserted relative to a node?
   - A) Right subtree
   - B) Left subtree
   - C) At root
   - D) Any place

5. Where is a larger value inserted relative to a node?
   - A) Left subtree
   - B) Right subtree
   - C) Root
   - D) Leaf

6. What is the time complexity of searching in a balanced BST?
   - A) O(1)
   - B) O(log n)
   - C) O(n)
   - D) O(n^2)

7. What is the worst-case time complexity of searching in a skewed BST?
   - A) O(1)
   - B) O(log n)
   - C) O(n)
   - D) O(n^2)

8. What is the time complexity of inserting in a balanced BST?
   - A) O(1)
   - B) O(log n)
   - C) O(n)
   - D) O(n^2)

9. What is the worst-case time complexity of inserting in a skewed BST?
   - A) O(1)
   - B) O(log n)
   - C) O(n)
   - D) O(n^2)

10. How is the minimum value in a BST found?
    - A) Leftmost node
    - B) Rightmost node
    - C) Root node
    - D) Any leaf

11. How is the maximum value in a BST found?
    - A) Leftmost node
    - B) Rightmost node
    - C) Root node
    - D) Any leaf

12. What traversal gives sorted order of elements in a BST?
    - A) Preorder
    - B) Inorder
    - C) Postorder
    - D) Level Order

13. What node is used to replace a node with two children when deleting it?
    - A) Root
    - B) Leaf
    - C) Inorder successor (or predecessor)
    - D) Any node

14. Can a BST have only left children?
    - A) Yes
    - B) No
    - C) Only root
    - D) Only leaves

15. Can a BST have only right children?
    - A) Yes
    - B) No
    - C) Only root
    - D) Only leaves

16. What is the root of a newly created BST?
    - A) First inserted node
    - B) Leftmost node
    - C) Rightmost node
    - D) Leaf node

17. What is an inorder successor?
    - A) Previous node in inorder traversal
    - B) Next node in inorder traversal
    - C) Root node
    - D) Leaf node

18. What is an inorder predecessor?
    - A) Previous node in inorder traversal
    - B) Next node in inorder traversal
    - C) Root node
    - D) Leaf node

19. Is level-order traversal dependent on BST property?
    - A) Yes
    - B) No
    - C) Only for balanced BST
    - D) Only for skewed BST

20. What is the height of a perfectly balanced BST with 7 nodes?
    - A) 2
    - B) 3
    - C) 4
    - D) 1
